## Taj Mahal Graphics Drawing

This is a graphics drawing project in C language that draws the Taj Mahal using basic graphics functions. It uses the graphics.h library to create shapes and lines to form the iconic Indian monument.

## Features

- Draws the Taj Mahal using basic graphics functions
- Uses the graphics.h library to create shapes and lines
- Allows users to modify the size and color of the Taj Mahal
- Very clear and easy to understand

## Requirements

- Turbo C compiler or any other C compiler that supports graphics.h library
- Windows operating system (tested on Windows 10)

## Installation

1. Download the source code from the GitHub repository.
2. Extract the files to a folder on your computer.
3. Open the Turbo C compiler or any other C compiler that supports graphics.h library.
4. Load the source code file and compile it.
5. Run the executable file to see the Taj Mahal drawing.

## Usage

1. Modify the size of the Taj Mahal by changing the values of width and height in the source code.
2. Modify the color of the Taj Mahal by changing the values of color in the source code.
3. Compile and run the executable file to see the updated Taj Mahal drawing.

## Known Issues

- The graphics.h library may not be supported by newer versions of C compilers.
